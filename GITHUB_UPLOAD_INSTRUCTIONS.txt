دليل رفع المشروع الكامل

هذه الحزمة هي مشروع المصدر الكامل وليست نسخة build.

يجب أن تكون الملفات التالية في جذر مستودع GitHub مباشرة:
package.json
package-lock.json
build.js
vercel.json
README.md

ويجب أن تكون المجلدات التالية بجانبها مباشرة:
public
scripts
test
.openai

لا ترفع المجلد الخارجي الذي يحتوي المشروع كمجلد فرعي داخل المستودع.
ارفع محتويات الحزمة نفسها إلى جذر المستودع.

أمر البناء العادي في Vercel:
npm run build

أمر البناء مع التحقق الكامل من 1,775 بندًا:
npm run build:verified
